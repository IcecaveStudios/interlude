public mixed invoke(callable $operation, integer|float $timeout = INF, integer $attempts = INF, integer|float $delay)

Repeat non-blocking operation until it succeeds, a timeout period is reached, or the maximum number of attempts have been performed.

The operation function is invoked with two parameters, the remaining timeout period (in seconds) and the remaining number of attempts, including this one.

The operation is considered successful if it returns a value that fails a strict comparison with false (ie, $value !== false).

Parameters

callable $operation The non-blocking operation to perform.
integer|float $timeout The maximum time to wait for the operation to succeed, in seconds.
integer $attempts The maximum number of attempts to perform.
integer|float $delay How long to delay between each attempt, in seconds.

Return Value

mixed The return value of the operation.

Exceptions

AttemptsExhaustedException If the operation is attempted the maximum number of times without success.
TimeoutException If the timeout is reached before the operation is invoked successfully.
