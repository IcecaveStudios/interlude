public mixed invoke(callable $operation, integer|float $timeout = INF, integer $retries = INF, integer|float $delay)

Repeat an operation until it succeeds, times-out or too many attempts are performed.

The operation function is invoked with two parameters, the remaining timeout period (in seconds) and the number of retries remaining.

The operation is considered successful if it returns a value that fails a strict comparison with false (ie, $value !== false) and does not throw an exception.

If the operation directly throws an exception that implements InterludeExceptionInterface the exception will propagate immediately, bypassing any remaining retries.

Parameters

callable $operation The non-blocking operation to perform.
integer|float $timeout The maximum time to wait for the operation to succeed, in seconds.
integer $retries The maximum number of retries to perform.
integer|float $delay How long to delay between each attempt, in seconds.

Return Value

mixed The return value of the operation.

Exceptions

RetriesExhaustedException If the operation is retried the maximum number of times without success.
TimeoutException If the timeout is reached before the operation is invoked successfully.
